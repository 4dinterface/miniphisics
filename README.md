Известные проблемы в этой версии
=================================
Ускорение от гравитации добавляется к вектору скорости перед проверкой на столкновение, и только после вектор скорости изменяет координаты.
Если коэфициент эластичности мяча равен 1, то мяч будет прыгать с каждым разом выше предположительно из за того что гравитация успевает увеличить ускорение до оотражения,
это дополнительное ускорение и увеличивает высоту прыжка
